## 📖 About the Project
It unites the people from same area or community. It is the solution for the Communities that are spread in whatsapp groups, it is for better connections and social networking.

---

## 🛠 Tech Stack
- Backend  : FastAPI + Jinja2
- Database : PostgreSQL
- Frontend : HTML + Plain CSS + JS (Jinja2 templates)
- Auth     : JWT + bcrypt
